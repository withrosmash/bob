# Bob — Daily Intelligence

Bob is Spencer's personal daily briefing app: weather for Canary Wharf, good news only, sector intel, one unexpected thing to do, a daily five-question game, and the diary. Vanilla HTML/CSS/JS, no build step, installable as a PWA.

**Live:** https://withrosmash.github.io/bob/

## First launch

1. Open the app. Bob asks for your **Anthropic API key** (get one at [console.anthropic.com](https://console.anthropic.com) → API keys). It's stored in localStorage on your device only and sent only to Anthropic's API.
2. That unlocks the briefing, pulse, convince and game features. Each makes at most one Claude call per day (cached until midnight London time; ↻ forces a refresh).

## Install on your phone

- **iPhone:** open the site in Safari → Share → **Add to Home Screen**.
- **Android:** open in Chrome → menu → **Add to Home screen** (or the install prompt).

## Google Calendar setup (~5 minutes, one-off)

The Calendar tab needs a Google OAuth Client ID of your own:

1. Go to [console.cloud.google.com](https://console.cloud.google.com) and create a project (e.g. "Bob").
2. **APIs & Services → Library** → search "Google Calendar API" → **Enable**.
3. **APIs & Services → OAuth consent screen** → External → fill the app name ("Bob") and your email. Under **Test users**, add your own Gmail address. (Staying in "Testing" mode is fine for personal use.)
4. **APIs & Services → Credentials → Create credentials → OAuth client ID** → type **Web application**.
   - Authorised JavaScript origins: `https://withrosmash.github.io` and `http://localhost:8000` (for local testing).
   - No redirect URIs needed.
5. Copy the Client ID (ends in `.apps.googleusercontent.com`), open Bob → ⚙ settings → paste it → Save.
6. Calendar tab → **Connect Google Calendar** → sign in.

Access tokens last ~1 hour; Bob refreshes them silently when needed.

## Local development

```sh
python3 -m http.server 8000
# open http://localhost:8000
```

No build step, no dependencies. Icons are generated by a small Python script (see repo history) — they're checked in, so you never need to run it.

## Architecture notes

- `js/claude.js` — direct browser calls to the Anthropic Messages API (`claude-sonnet-4-6`) using the `anthropic-dangerous-direct-browser-access` header. News features use the server-side web search tool for real, current stories; the game uses structured output (JSON schema).
- `js/storage.js` — localStorage helpers plus a per-day cache keyed by the London date, so each feature costs one API call per day.
- `sw.js` — caches the app shell for offline/instant startup; never touches cross-origin API requests.
- Everything uses relative paths so it works at `withrosmash.github.io/bob/`.

## A note on the API key

This is a personal tool. The key lives in your browser's localStorage — don't enter it on shared devices, and rotate it at console.anthropic.com if you ever suspect it leaked.
